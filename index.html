1

Skills as NFTs

Each AI skill is minted as a Metaplex NFT. Ownership grants your agent the ability to execute that skill. Transfer, sell, or compose skills freely.

2

Mobile-Native Distribution

Integrated with Solana dApp Store on Seeker. Push notifications when new skills match your agent's profile. One-tap install from lock screen.

3

Agent Autonomy

Agents browse SkillDock autonomously — acquiring skills they need to complete tasks. Spending is governed by user-set SOL budgets.

4

Creator Economy

Skill creators earn SOL on every install. On-chain royalties via Metaplex. Revenue sharing for composable skill bundles.

5

x402 Agent Payments

Agent-to-agent skill purchases via x402 payment protocol. Your agent can pay another agent's skill fee autonomously — enabling a true agent economy.
